# Golf Rivals Logo Usage

## Logo Variants

**primary:**
Use for hero sections, login/register screens, brand manual and large brand moments.

**horizontal:**
Use for desktop header, footer, presentations and places where there is enough width.

**icon:**
Use for favicon, app icon, small buttons, mobile nav, compact headers and loading states.

**monogram:**
Use for mobile headers, profile areas, watermarks, premium badges and places where the full wordmark is too large.

**white:**
Use on dark complex backgrounds when the gold version loses contrast.

**gold:**
Use for premium moments, brand manual examples, splash screens and high-value highlights.

## Recommended App Usage

**Home:**
Use primary or horizontal.

**Global layout/header:**
Use icon on mobile.
Use horizontal on desktop if there is enough space.

**Login/Register:**
Use primary.

**Brand Manual:**
Show all variants.

**Profile:**
Use monogram or icon.

**Live Game:**
Use icon or monogram only. Do not use the full wordmark because it wastes space.

**Summary:**
Use horizontal at the top and icon/monogram in compact sections.

## Rules

- Do not stretch the logo.
- Do not recolor manually with CSS filters.
- Do not use full wordmark in tiny areas.
- Do not place gold logo on a busy gold background.
- Do not mix old /brand-logo.svg references once the new logo system is complete.
